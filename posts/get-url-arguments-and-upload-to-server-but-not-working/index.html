取得URL參數值、上傳伺服器沒有反應問題

取得URL參數值

安裝JQuery外掛：url jQuery Plugin，即可直接用$url() 取得網址參數值。

上傳伺服器沒有反應問題

今天遇到一個很奇怪的問題，在本地端都可以執行 JS ，結果上傳到伺服器都無法作用。後來查一下原始碼，原來是 Cloudflare 啟用 Rocket Loader ，導致所有 JS 都無法正常啟用。

之後在JS段落全部增加 data-cfasync="false" ，讓Cloudflare在這檔案關閉Rocket Loader，程式碼就可以正常運作了！

<script data-cfasync="false" src="..."></script>
