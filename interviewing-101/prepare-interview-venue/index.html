面試場地體驗大不同 #

我朋友有不同面試空間體驗。A 公司是在嶄新辦公大樓裡，辦公區是氣派開放大空間，但卻和面試官擠在狹小會議室，兩人相隔約一隻手臂，進行時略感尷尬。B 公司則是到場後，隨意在公司大廳找個圓桌和他進行面試。C 公司面試時，進場動線和辦公區完全分離，帶他進到兩人適中大小會議室。想當然爾，在 C 公司面試體驗最佳。

面試場地對面試體驗不容小覷，整潔、舒適環境，可以讓面試者專注，不會受到環境影響分心，更能發揮實力；反之，如果面試者一進公司對環境感到不安，連帶影響他們面試表現。

場地準備就緒 #

空間大小適中 #

選擇一個大小適中場地，可以讓面試官與面試者都有一定伸展空間，彼此可以舒適交談，而不是侷促不安。

單人面試，小型會議室即可，不用浪費到大型會議室。你可以和同事兩人坐定測試，確認彼此間隔適當、交談無障礙；多位面試官面試，就需要大一點空間，準備桌椅數量可以容納所有面試官以及面試者。

團體面試，通常是面試者數量大於面試官。此時可以安排等候區、休息區，盡可能安排座位，讓面試者可以稍作休息準備，並且有置物空間可以安放器材，例如筆記型電腦、隨身包包等。

面試場地內的桌面大小安排，至少是比紀錄工具稍大桌面，例如可以擺放 A4 紙、筆記型電腦記錄過程，並再加一杯水空間。

提前預約使用 #

選定空間後，先行預約，確保面試當日使用無虞。公司會議室通常行程緊湊，盡可能提前預約會議室是必要的，以及為了避免無心占用情況，可和內部共享會議室使用時間，避免使用衝突。

使用當日若前手為外部會議或重要會議，例如和客戶訪談或者是主管會議，可以提前提醒會議主持人協助注意會議時程，或是當日改為使用其他會議室，避免延宕面試時間。

確認設備準備就緒 #

面試有時候會需要使用器材，事先確認借用與占用情況。例如螢幕、投影機、螢幕轉接線材、筆記型電腦、充電設備、麥克風、視訊鏡頭、簡報筆、白板、白板筆、訪客用 WIFI 使用資訊等等。

在面試當日，確定設備運作正常、確認網路連接穩定性。必要時可以準備多組設備，有突發狀況時隨時支援。

準備緊急聯絡人 #

人有失手，馬有亂蹄。即使萬事俱備，偶爾免不了發生意外。可在事前安排技術或行政支援緊急聯絡人，可以幫你迅速排除狀況，例如尋找支援人力、借用臨時器材或是排除技術障礙等等。記得事後紀錄疑難排解過程，下次遇到時不再慌亂。

準備面試場地時，記得挑選環境適中，讓面試雙方都有舒適體驗，並確保空間可用性、設備完善，減少當日突發狀況風險。以及準備好緊急聯絡人，有意外時可以緊急排除，保持面試流暢順遂。
