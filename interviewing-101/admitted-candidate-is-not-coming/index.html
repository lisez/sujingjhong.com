我找到更好的工作了 #

除非你所在公司或產業，是屬於強勢公司選人市場，例如鼎鼎有名的尖牙公司(FAANG)；不然多數情況，你可能會因為各種原因，從人資單位收到面試者婉拒信。這還不打緊，更可能是在報到前一刻，才收到錄用者不克前來。所以你仍須應對此些狀況。

從面試者角度，無論是婉拒信，或者報到前反悔，不管理由是什麼，隱含不言說的都是：「其他公司比你好。」 你也不用感到心寒，因為公司也是從一批求職者挑選出最適合對象。這是雙向選擇過程。

對面試官而言，就是對婉拒情況作好準備。

作好沙盤推演 #

認清公司市場位階 #

公司在市場上位階，這是很現實問題。思考這問題會讓人很不舒服，你必須穿過公司宣傳屏障，誠實地從外部思考公司在產業上、市場上，是屬於什麼等級。你所在公司，在產業中不同等級，用人標準自然也就不同。

如果公司是 FAANG 之流，你要做的就很單純：「找到真正優秀的人」，也不太需要多加思考求職目的，說不定就職後貴公司名片就是職涯聖杯；倘若答案是否定的，而真的遇到一位無比耀眼候選人，那你要認真思考求職者應徵動機為何，說不定會開始懷疑他是不是沒有自信？是不是只想當跳板？會不會沒發現自己的價值？或者很單純只找個離家近公司。

進而你會開始思考，**你「自己」為什麼會想在這工作？為什麼他會想要和你一起工作？**接連去評估公司、團隊、職位，真正適合的人才定位是什麼。並對每位職位候選人評價到職機率為何，尋找適合公司現階段人才，並且可能到職的候選人。

排定人選繼續面試 #

當你有數位候選人時：

  1. 排定優先序：你可以根據面試表現情況，排定候選人順序，讓人資單位可以按優先序通知；如果你認為都差強人意話，也可以從缺，進行下一梯次面試。
  2. 說明爭取程度：你可以和人資單位進一步討論，需要多積極爭取候選人。資源預算有限，不太可能每位候選人都投注相同程度爭取。況且有時候需要動用公司資源，例如第一年就給予該職務最高等第，需要進行特別流程，此時你會需要和上級說明為何這位人才值得。

如果沒有其他候選人遞補，那也只能等待下一輪面試。

重新思考候選人評價 #

當有位候選人表達婉拒時，而你想積極爭取時。你需要先評估這位候選人有多特別：

  1. 本身面試表現如何？
  2. 和其他人選差異在哪？
  3. 有什麼不可取代性？

也許你會想，想必是面試表現不錯的候選人，才會通知錄取。當他婉拒時，則正式另一個好時機，從當初熱情消退，理性上揚，由後設性角度，重新思考這位選人表現如何。給彼此一個重新評估機會。

思考完這位候選人表現，如果同期有其他候選人，重新思考在這些候選人中等第如何，相較於其他候選人，他的差異點是什麼？ 是剛好在同期候選人表現突出？或是只有他解出題目？或是只有他用新方法？重新回憶思考，他和其他人比較差異在哪，才讓他得以雀屏中選。

接著整合上述兩個問題答案，重新思考他的面試表現以及差異點，這位候選人有什麼不可取代性，且足以讓你相信，過了這個村就沒這個店，時機難得，一定要好好把握。是他的人格特質對於職務內容是萬中選一？或是他獨特技能組合剛好是公司急需人才？或是有數年特定產業經歷剛好是下個產品著陸點？重新將這位候選人，以不同視界觀察他的表現、文化契合度、人格特質或技能組合等等，以更寬廣面向思考，到底是具有什麼獨特性，讓你難以放手。

思考完以上這些問題，如果答案都是肯定的，記得記錄下來。因為之後你將需要用來說服人資、用人主管、你的上級或是其他同仁，你將化身成說客說服求職者，也需要轉身成為經紀人，打通在公司內道路。

協調擊破婉拒原因 #

不過大部分情況，你只需要整理上述這些資訊，化為爭取動機，並和人資或行政單位討論，面試者婉拒理由是什麼，從理由去推敲面試者婉拒背後的原因需求為何，以及在公司制度框架下，可以給予到什麼程度。

主要原因需求可分成：

  1. 物質需求：薪金、設備等物質性需求。例如想要多點薪水、租金補貼、通勤補貼、公司提供設備等等。
  2. 勳章需求：和別人不一樣的需求，部分時候會和物質需求疊合。例如獨立辦公室、要求「資深」職稱、不需要每天進辦公室等等。
  3. 工時需求：個人化上班時間或是假期安排等需求，例如彈性工時、下午四點半前離開因為要接小孩、固定時間要在家照顧長輩等等。
  4. 職涯需求：看重自身職涯發展，可能期待有到職培訓、教育訓練、導師制度，或是想找與自身興趣、職涯有關的職務內容。
  5. 到職前需求：另一種可能是因為在找工作前，已經有其他安排，例如安排要出國玩、要完成學業，因而選擇了一間可以彈性調整報到時間公司。

至於要怎麼發現婉拒原因，則仰賴面試時去探索求職者公司選擇因素，以及在聯繫時多問一些，必要時直接直球：「其他公司比較吸引你的點是薪水還是其他什麼部分呢？」

了解後，如果真的很喜歡此位候選人，可考慮在公司制度框架內，給予最大彈性。必要時可以適時表現誠意，例如表示會寫明在契約書內，或是在來信內保證，又或是找到職後主管背書。讓求職者足以相信，你不是空口說白話，是會確保承諾履行。

並且你心中要作個最壞打算，也許到任表現不如當初預期。 畢竟你僅能由面試中片段資訊，去推敲可能能力以及表現落點在哪，實際情況，只有到職後才知道。所以有可能到職後，和面試時判若兩人，表現讓你跌破眼鏡。如果你仍覺得一試，那就試試。

不會是最後一次 #

在面試官職涯，中意者不來屢見不鮮。有是一個月前敲定，結果到職前一日甩手不來；也有壓後到職日期，但實際上卻是去其他公司到職。各種狀況層出不窮。你可能會第一次遇見，但絕對不會是最後一次。 **所以還是回到開頭說的，面試是雙向選擇過程，你在選擇誰能錄取，面試者也在選擇他要去哪裡。
