面試來到收尾階段 #

一場面試終於來到了尾聲，通常你已經對求職者能力、文化契合度以及人格特質有一定了解；又或者你覺得面試可以告個段落，為這場面試落下終幕，就可以進入面試收尾階段。

當你在準備考試、唸書或學習時，一定會覺得開頭和結尾記憶最鮮明，這是心理學上的開頭效應以及最新效果所影響。一樣的，在一場面試中，除了中途有記憶亮點外，多半也是對於開頭以及結尾記憶最為清晰。所以在面試尾聲，你也是要作好準備進行。

送客與交流 #

總結性回答以及聯絡方式 #

當面試官提問環節進行一段時間，就可以由此場主軸面試官作出決定，決定是否要結束這場面試：

  1. 時間因素：面試最好是直接框定時間，可以讓面試者了解時間成本，以及避免面試官無止盡提問，或不能有效充分地利用時間。例如一開始就可以和面試者說明須一或兩小時，接著在時間快到時，裁示進入收尾階段。
  2. 無繼續必要：例如面試提問與回答陷入僵持，或者面試官認為已可充分評價，依臨場情況適時收尾，不用再等到表定時間結束。

在收尾階段，可以先詢問其他面試官是否還有要詢問其他問題，確保每位面試官都已詢問完畢。

接著你可以先做個總結性回答，例如主動進行公司介紹、組織架構、福利制度、職缺職務工作細項說明，常見問題也可以一併主動告知。此方式可事先準備常見問題，並得以明確及一致地回覆，避免因準備不周，導致說明時遺漏細項，而在不同面試者間有不同回答。同時確認公司招募廣告與說明事項是否一致，預防廣告不實情況，而受到主管機關裁罰。

總結性回答，可以有效地降低面試者提問。畢竟提問內容大多是每日工作內容、福利制度、請假制度、工作流程等等。你可以站在求職者角度，思考加入一間公司，你會問什麼問題。

最後可以提供面試者提問環節，釐清面試者疑惑，或者有進一步需要了解之處。同時收集面試者提問，有反覆出現問題，可在之後面試中補充說明。面試者提問，會反應出關注點、對職位及公司理解程度，例如可能詢問是否需要值班、加班程度、平常怎麼討論等等，都可以進一步側寫他在乎什麼。

此外記得確認彼此後續聯絡方式，例如一律由人資溝通，或是會透由獵人頭處理。說明等待回覆期中，可以怎麼聯絡、可以協助哪些事項，保持雙方間溝通管道暢通。

送客與場外交流 #

當所有事情告一段落，記得要儀式性地宣布「面試結束」，表示從現在開始就不屬於面試流程。這有個好處是，雙方可以輕鬆角度進行場外交流、閒話家常，如果是用餐時間，可以介紹附近可以去哪用餐，或是口渴可以去哪購買涼水飲料。

不介意話，也可以帶著面試者介紹公司環境、設備，同時介紹公司歷史、文化理念等等，以導覽方式向求職者介紹公司，讓面試者了解實際工作環境時，同時作品牌宣傳。對公司都是絕佳曝光機會。

和其他同事交流 #

在每場面試結束後，你記得須留下一小段時間，和其他面試官快速討論，趁記憶鮮明時，釐清面試中事物。如果無法即時討論，也可以寫成筆記，方便日後回顧，同時反思自己表現。

面試結束後，其他同事多半會好奇面試者表現如何。此時作為一位面試官，你需要拿捏自己保密範圍。

  1. 保密求職者講述內容。面試過程中會涉及能力評價、自我評斷、人生故事歷程等林林種種諸多面試者自我內容，而你需要依此內容作出面試評價決定是否錄取，這些內容是只有在行使面試官職務時，才能接觸到，所以我自己會認為屬於業務上祕密，不得任意洩漏。
  2. 對於公開內容，不多加評議。現在從網路上收集資料稀鬆平常，甚至許多求職者，甚至我自己，都會將學經歷公開在例如 Cake, LinkedIn 等社群網站，取得公開內容並非難事。不過面試官有篩選履歷、評價內容權限，對公開內容評價，無疑是公開心證，表示出特定看法，而因此間接評價特定面試者。我認為此種評價，一樣屬於面試官業務上祕密，所以不宜多加議論。

另一種則是面試中面試官趣事，畢竟這種事情就是公司軼事，面試官彼此間就是戰友，得以彼此回憶拉近感情，同時調劑面試後壓力，好好紓壓一番。

終於結束一場面試 #

面試對於面試者以及面試官，無疑都是場耐力賽，雙方都需要出兩百分心力與精力，才能讓面試完美落幕。所以即使在面試尾聲，也要多加留意，可以進行總結性回答減少面試者疑惑，並確認聯絡方式讓彼此溝通暢通，有時間也可以進行公司導論，讓潛在候選人了解公司，增進到職機率。最後記得留些時間和其他面試官交流，無法的話也記得事後盡速留下筆記。以及對於談論面試者評價，務必謹言慎行。
