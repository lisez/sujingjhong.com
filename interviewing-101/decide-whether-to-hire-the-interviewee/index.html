錄取或不錄取，是個問題 #

面試完後，不管你用什麼方式評分、評價，你最終一定要做出決定：「是否錄取？」不論你做出什麼回覆，例如面試者表現如何、履歷如何、談吐如何，繞了一圈後，還是需要回答這一翻兩瞪眼的問題。錄取或不錄取，是個問題。

面試者表現令人驚嘆或是差強人意，這種極端值表現，相信你一定會很快作出決定；但通常是介於二者之間，就會需要反覆思量。另種情形則是數位面試者間比較時，該如何選擇。

這時候就常出現幾種句法：

  • 如果 A 面試者有（優點）／如果 B 面試者沒有（缺點）……
  • 要不是 C 面試者有（缺點）／要不是 D 面試者沒有（優點）……

難以抉擇的候選人，通常會有些缺點或是不具備某些優點，可以此作為思考座標起點；另一種是放眼未來，留意對方有沒有成長心態，期許將來有其他長處或是降低短處影響。

向自己提問 #

是否可容忍缺點？ #

人非聖賢，孰能無過？即是聖賢，又誰無過？凡是生而為人，一定會有缺點。況且人天性多少會隱惡揚善，又有誰會大聲嚷嚷自己缺點？在面試中也是一樣，面試官多半會問求職者缺點，不管是直球詢問或曲球對決，敏銳面試者或是坊間面試技巧，也一定告訴你要包裝自己的缺點，所以多半回應模式是：

  1. 我曾經有個缺點，但我做了改進與調整，我現在更好了。
  2. 我有個缺點，但實際是個優點。
  3. 我有個缺點，但對你不是缺點，反而是優點。

在面試中盡可能聽出弦外之音，也是你的責任，例如追問缺點的定義、追蹤改進方法或是詢問那什麼是你的優點等等。在面談後，可以和其他面試官討論，面試者優缺點是什麼、他可能會在什麼情況造成影響，例如可能會和你激辯、或者不表示看法等等。

並且要建立一個對缺點認知，江山易改本性難移，是錯的；而是將缺點類比成一種惡習，既然是習慣，就可以觀察行為，給予建議調整。即使如此，改變仍需要時間及無比耐心。你必須在這段時間內，循循善誘以及接受惡習，並在內心期許每天都會好一點。

因此在這段時間內，你必須「容忍」習慣。問問自己，可以接受並容忍對方的缺點嘛？如果不行，那原因是什麼；如果可以，原因是什麼。這些問題都有助於釐清團隊價值觀、職務需求，以及最終會反射回你自身、探索自我內心，了解自我對於人際缺點限度範圍以及偏好原因為何。

是否有獨特之處？ #

憲法上會提到一個概念，就是保護人性尊嚴。而人性尊嚴的核心價值，就是尊重個人的自我決定權，任何人都可以享有同等自由，追求自我理想生活，成為心底那個人。所以人之獨特性，對於一個人而言，是他在追求自我時的重要表徵。

在面試中你可以觀察，求職者在講述什麼時神采奕奕或是萎靡不振。可以從講述層面、事物、用詞等，或是繼續追問，了解這位面試者心之所向，又或者什麼讓他痛苦疲憊。從這些細節，了解獨特之處以及適不適合應徵職務，這是用人穩定性影響因素之一。

而對於獨特評估要要思考是：

  1. 劍走偏鋒，出奇制勝：獨特性極為搶眼，會構成優缺兩面刃。例如善於溝通也可以說長於激辯，要思考獨特性的一體兩面性，並回到如果變成缺點是否可以容忍情況。
  2. 重劍無鋒，大巧不工：獨特性表現在舉手投足間，其優缺都不太明顯，在此階段多半已降低獨特性短處，並強化長處，例如會選擇性激辯。
  3. 草木皆劍，無劍勝有劍：將自身獨特性修煉大成最高境界，可遇不可求。

是否持續自我提升？ #

你可能聽過一個故事，就是一位智者和國王要白米，第一天兩粒，第二天四粒，國王不以為意，故事結局你應該知道了，國王最後要送出上百萬粒甚至上億粒米。當講到複利效應時，多半是提這個故事。

延伸到自我成長，大概就是每天進步萬分之一、千分之一，一年後你已經進步多少的心靈雞湯；或是每天退步萬分之一，一年後你已經退步多少的警世故事。但總之，這些故事核心只在於持續小改變的重要性。

轉換到專業職涯，也許這位求職者現階段表現欠佳，但他卻有持續的精進計畫，只要預期堅持繼續，士別三日，必當刮目相看；但如果他沒有自我提升計畫，也許就停滯在這。面試時一個考量點，就是觀察他有沒有持續精進計畫，例如每天寫些小程式、閱讀一本書；也可以是和職涯無關的，例如擬定計畫攻克百岳。

觀察點在於，對於一件生命中的事物是否有推升的計劃。只要他有這個觀念，那對於推進職涯，就只是關注點以及時間點問題，也許只是現在沒有資源、沒有時間或是沒有誘因等等，但是只要有這個價值觀、而不是維持現狀，這樣就夠了。

你願意和他一起工作嗎 #

巴菲特表示過，和不喜歡的人共事工作，就像是為了錢結婚一般。無論如何都不是個好主意。

所以思考完上述問題後，最後一個就是：「你願意和他一起工作嗎？」想像一下當你投下同意票後，你願意和他一同工作嗎？

這是敝公司在評選時，一定會問到的問題。一個很單純的是或否，如果答案是否，那就不用再接續討論了。因為你打從心底是排斥的。況且工作時間表定占用你一天三分之一，會是你人生中重要時光。而身為面試官，卸下身分時，也是公司成員之一，你可能業務或平時，就會與錄取者互動，何必讓自己痛苦呢？
