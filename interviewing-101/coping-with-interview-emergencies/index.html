來個措手不及 #

準備再怎麼周全，臨時狀況仍措手不及。目前有遇到三次印象深刻突發狀況：

  1. 借錯會議室：那時我公司開始推行上機考，需要了解作題狀況。結果借到一間沒有螢幕會議室，還好在面試半小時前意識到這問題，趕緊搬螢幕過去，讓面試可以順利進行。
  2. 面試者電腦接不上：當時求職者使用自己設備面試，公司提供的 USB Hub 和面試者電腦相性不良，連忙找了其他組 USB Hub 試試看，還好最後有成功配對到一組。這過程讓面試延後了 10 分鐘左右才開始。
  3. 公司設備無法連網：有次提供公司電腦給面試者使用，前天測試一切正常，結果當日一早測試機器時，發現網路模組失效，連試半天一樣無法作用，但又沒有其他機器可以使用。最後是面試者他可以自行用手機連網。之後公司就有多一台備機，避免此種情況發生。

面試時偶有狀況，你需要臨機應變處理，有些是可以排除，讓面試繼續進行；有些則需要當機立斷。以下就說明一些常見狀況。

面試突發狀況 #

面試者沒在預定時間抵達 #

最常見就是面試當天，面試者沒有在預定時間抵達。主要情形可以分為以下六種：

  1. 邀錯時間：一開始就邀錯時間了，導致面試者前來時沒有人知道，或是到預定日才發現邀錯時間。
  2. 提早到：在當日提前 15 分鐘以上抵達面試地點。
  3. 遲到：在當日預定時間後才抵達。
  4. 改期：當日才說明希望能將面試改到其他時間。
  5. 取消：當日才通知要取消面試。
  6. 消失：時間到了人沒出現，後續也沒有任何消息。

邀錯時間很明確是公司失誤，如果是當日前來時才發覺邀錯時間，應對方式主要有下列幾種。但無論哪種，都仰賴現場人員臨機應變，盡可能降低傷害。

  1. 立即安排面試：你和其他面試官需要立即動員所有資源，挪出可用地點、設備，立即現場安排面試。
  2. 當日其他時間安排面試：如果沒辦法立即安排，可以和面試者協商其他時間進行，避免白跑一趟。可以考慮安排一些茶水點心作為補償。
  3. 安排其他時間面試：和面試者說明邀錯時間狀況，可行的話立即磋商下次面試時間。可以考慮補償來回交通費。

如果是提早到，你會需要寒暄並安排休息空間，讓面試者稍作休息準備，或是可以請他到附近繞繞再進來。要注意是，不要將面試者晾在現場，讓他不知所措。

另一種則是遲到，面試遲到雖然是大忌，不過時有發生。有提前說明，就看能不能接受遲到理由與等待時間，或是直接另行安排面試時段。

改期的話，就是不變應萬變。評估狀況看是否接受面試者理由以及時間，或是另約時間進行，決定後再聯絡對方。有時面試者是試探性詢問，不是真的想改期，也許稍後聯絡就會如常進行，不用改期。

另一種就是當日取消。既然對方都開口取消了，能怎辦呢？公司可以稍微詢問取消理由，通常八九不離十是有心儀工作，但面試者通常不會明說。公司此時最忌諱追問細究，有一種情況例外，就是你很喜歡這位面試者，可以給他相當或更高報酬。

最後就是消失，當日 No Show，這位面試者自始就在你人生中消失了。基於關心、同理心，可以當日稍後或晚幾日關心一下狀況，同時也可以放下心中大石。

面試官沒在預定時間抵達 #

面試官也可能會未在預定時間抵達。面試官通常是內部職員，以及時間是預定好，所以相較於面試者，情形主要為：

  1. 記錯時間：聯絡時間沒有失誤，單純是自己記錯時間。
  2. 遲到：可能因為當日前手行程延宕，或是通勤時間影響等等，導致未能如期出席面試。
  3. 取消：因為諸多因素，當日無法出席面試。
  4. 消失：不知為何人就不見了。

記錯時間這是面試官個人失誤，避免方式就是當日提醒時間，並了解面試官行程狀態，推估他通勤時間，確認是否出發，以及是否能如期抵達。盡量要求面試官提早抵達面試地點，作好準備。

面試當天有其他行程影響面試，導致遲到。盡可能提前告知行政人員或其他面試官，可以做一些緩衝性安排，例如調整流程、告知面試者會延後時間，或是說明有多階段面試。如果面試已經在進行中，通常 5-10 鐘左右時間內進場，還算是可接受範圍；超過時間，請由其他同仁代為面試，或是安排下階段流程。

如果當日真有突發狀況導致無法出席，盡可能提前告知其他面試官或人資單位，讓他們可以協助調整面試流程，或是重新安排面試時間；盡可能不要面試快開始時，才通知無法出席，我想其他同事應該會想把你千刀萬剮。如果是一人面試官，則盡速提早告知面試者另行安排，避免白跑一趟。

天有不測風雲，人有旦夕禍福。如果面試官真的消失了，那就是公司重大事件，務必要找到人，確認發生什麼事。同時間緊急協調其他同事進行面試，或是告知面試者改期或取消。

面試官通常是內部人員，所以你可以事前提醒，面試快開始前確認出席狀況，必要時啟動應變措施。

網路、設備出問題 #

網路設備等，在面試前作好確認，確認是否運行如常。倘若發生突發狀況：

  1. 尋求支援：越緊急越需要交給專業，立即尋求支援人員幫助排除。
  2. 尋找備品：如果是設備出問題，看公司現場是否有其他備品可以使用，或是商借現場同仁設備，不過務必要確認資安政策，確保機密不外流。
  3. 替代方案：例如更換流程、地點、順序，直接轉換至沒有問題狀態是上策，或是有讓其他人有時間進行狀況排除，等排除後再一如往常進行。
  4. 授權面試者：如果公司政策或面試官接受，可以考慮讓面試者使用自身設備或網路，讓面試可以繼續進行。

突發性人身狀況 #

健康資料是個人資料保護法之特別保護之特種個人資料，所以除非當事人親口說，否則你是很難得知他人身心狀況，此時如果在面試時遇到例如顛顯發作、失去意識等等，會讓你難以招架。

  1. 平日準備好醫藥箱、AED 設備等，也需要作好人員訓練，可在短時間內取得設備或進行 CPR。
  2. 盡可能第一時間播打 119 或 110 尋求救助；位置不佳時則播打 112。並尋求專業救助指示。
  3. 人員充足時，可安排人員引導救助人員到達，增加救援機會。

臨場應變來自作好準備 #

其他如地震、颱風、火災等情況就不多做說明，人身安全保障第一，如果公司面對此種情況，仍繼續要求面試，你則需要思考這是否為安全就業環境。留得青山在，不怕沒柴燒，只是面試，不用賭身家。

所有臨場應變，多半來自事先準備。準備雖然無法萬全，但可加強你臨機應變技巧，讓你有多技巧得以組合應用。不怕一萬，只怕萬一。
