面完了還是不知道面試官是誰 #

面試者面完後，不知道面試官情形履見不鮮。朋友去了一間公司面試，面完只大概記得面試官是用人主管，至於姓名、職稱一概不知。另一位朋友，之前因為面試其中一位面試官時臨時有事，之後線上面試二面，全程沒開鏡頭，讓開鏡頭的他顯得尷尬。

面試官除了評估候選人外，也是公司形象窗口，是讓面試者留下印象第一線。開始面試前，你可以直接遞上名片，簡單介紹一下自己。

開始前請遞上名片 #

確立商務場合儀式 #

公司名片設計承載著企業形象，呈現企業象徵的顏色、符號、標語等，同時提供聯絡資訊，例如公司名稱、地址、統編，到個人聯絡方式，例如姓名、職稱、電話或電子郵件。小小卡片，背後承載千言萬語。

遞名片，已經是現代商務場合基本橋段，藉由這資訊卡片建立起溝通橋樑。在一個正式場合中，與會者要如何彼此破冰？最簡單方式就是介紹自己，接著遞上名片。這在職場上、商務上以及全世界，已經是基本儀式。嚴謹者會有一套遞名片以及收名片流程，例如面帶微笑、將名片轉至正面對向對方等等。

所以在面試時，你可以借用習慣威力，藉由「遞名片」雙手奉上名呈現的儀式感，宣告現在是正式商務場合。同時也隱含現在是代表公司出席，表彰公司形象，你的一言一行都和公司利害相關。

寒暄與友好關係起頭 #

名片上通常會有職稱、姓名，讓面試者了解面試官角色。遞上名片時，除了介紹自己外，同時也會說個幾句話。這時就是寒暄、建立友好關係開頭的好時機。如果面試在午後，可以詢問對方吃飽沒；或是詢問對方如何前來；也可以和面試者聊一下天氣如何。

簡單聊幾句，舒緩面試者緊張，讓面試者感受你的人情味，而不是面試官身分。此時對你和面試者，都是很好的準備期，當雙方都準備好了，再正式進入面談。

建立人脈資源 #

面試是個雙向選擇過程，公司從候選人中選擇，面試者也在公司間選擇。面試官此時就是公司代言人，你的專業表現與形象，也會烙印在求職者印象中。即使沒有錄取，但他可能在未來某一天變成你的人脈資源。

線上面試請開啟鏡頭 #

線上面試時，無法實體接觸到面試者，取代方式就是開鏡頭以及介紹自己。讓面試者了解你也是真實存在，看到表情、肢體而不是單純只有音訊，會讓面試者感受到更多真實性。

此外面試者總是會被要求開鏡頭，所以面試官也開啟鏡頭，會讓面試者感受到平等與真實性以及尊重。就如同看見面試者可以提高面試官信任度，面試者看見面試官可以提高對公司以及面試官信任度。

請你線上面試時，務必開啟鏡頭。如果是因為設備問題，請主動告知面試者你無法開啟鏡頭，表示你有在意這個問題。

勇敢面對面試者 #

你必須認識到，面試是一個雙向選擇互動，你更坦然面對面試者，更可以表現出你的自信與專業，而不是隱藏在面試官身分下的某人。在開始面試時，請準備好名片遞給面試者，同時簡單介紹自己，稍微寒暄下，讓面試在一段友好氣氛中準備就緒。
